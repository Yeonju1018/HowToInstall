3. <img width="338" alt="스크린샷 2022-12-26 오전 11 43 46" src="https://user-images.githubusercontent.com/48478079/209492952-be1522a0-0b1a-4b58-bf68-3ac9ac417a95.png"> 에서 "$ brew install git" 복사합니다.
